### Sorting Algorithms

## Time Complexity

* Insertion Sort
    - Worst:    O(n^2)
    - Best:     O(n)
    - Average:  O(n^2)
* Bubble Sort
    - Worst:    O(n^2)
    - Best:     O(n)
    - Average:  O(n^2)
* Merge Sort
    - Worst:    O(nlogn)
    - Best:     O(nlogn)
    - Average:  O(nlogn)
* Quick Sort
    - Worst:    O(n^2)
    - Best:     O(nlogn)
    - Average:  O(nlogn)
* Quick Select
    - Worst:    O(n^2)
    - Best:     O(n)
    - Average:  O(n)
